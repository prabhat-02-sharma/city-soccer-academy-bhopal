# City Soccer Academy Bhopal

Professional grassroot football coaching for kids aged 6-17 at Campion School, Bhopal.

## Tech Stack
- Next.js 15
- Tailwind CSS
- Lucide React
- Supabase

## Getting Started
1. Clone the repo
2. Install dependencies: `pnpm install`
3. Run development server: `pnpm dev`
4. Build for production: `pnpm build`
